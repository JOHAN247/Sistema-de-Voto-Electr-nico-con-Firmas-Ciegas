# Sistema-de-Voto-Electronico-con-Firmas-Ciegas
Aplicación desarrollada en Streamlit que implementa conceptos de Matemáticas Discretas y criptografía mediante firmas ciegas (Blind Signatures) para simular un sistema de votación electrónica completamente anónimo, garantizando la privacidad del votante y la integridad de los votos.
